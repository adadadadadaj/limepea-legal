LimePea

LimePea is a multipurpose Discord bot built for simplicity and power.
With features like:

  • ✅ Reaction Roles and Auto Roles
  • 🎉 Welcome Messages & Embed Setup
  • 📊 Leveling and XP Leaderboards
  • 🎫 Ticket System with Slash Commands
  • 🧠 Games like Tic Tac Toe and Fortune
  • 🛠️ Moderation tools like Ban, Kick, Timeout
Whether you're building a cozy community or a powerful mod hub, LimePea does it all.

It handles:
Reaction Roles, Welcome Messages, Leveling, Tickets, Embeds, Moderation, and Games — all in one clean package.
