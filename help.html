LimePea Help Menu

Here’s a full list of available commands and features you can use with the LimePea Discord bot.

🎮 General Commands

  • /help — Shows this help page.
  • /balance [user] — View your or another user's lime balance.
  • /give [user] [amount] — Give your limes to another user.
  • /toplimes — View the top 10 lime hoarders.
  • /daily — Claim your daily lime reward with streak bonus.
  • /gamble — Gamble your lime savings away.

🔨 Auction Commands

  • /auction_start — Start an auction.
  • /auction_bid — Bid on an auction.
  • /auction_end — (Admin)End the current auction.
  • /auction_status — Check the current auctions status.

🔧 Admin & Setup

  • /reactionrole_setup — Setup a reaction role on a specific message.
  • /reactionrole_remove — Remove a specific reaction role.
  • /autorole_add — Automatically assign roles when members join.
  • /level_toggle — Enable or disable level tracking in your server.

📊 Leveling & XP

  • /rank — See your current level and XP progress.
  • /leaderboard — Show the top XP earners in the server.
  • /set_level [user] [level] — (Admin) Manually set a user's level.
  • /lock — Lock the server.
  • /Unlock — Unlock the server.

🎫 Ticket System

  • /ticket_setup — Create a ticket panel.
  • /close — Close the current ticket.
  • /add [user] / /remove [user] — Manage members in tickets.

⚠️ Moderation

  • /ban [user] / /kick [user] / /timeout — Core moderation tools.
  • /warn [user] — Issue a warning to a member.

⭕ Tic Tac Toe

  • /tictactoe [user] — Challenge someone to a game of Tic Tac Toe.
  • /ttt_resign — Resign from a current game.

🧠 Games & Fun

  • /fortune — Receive a random fortune message. 🍀
  • /roll [sides] — Roll a dice with any number of sides.
  • /limefact — Get a fun lime-related fact.
  • /rate [thing] — Get a random rating for anything.
  • /battle [user] — Battle another user and see who wins.
  • /8ball [question] — Ask a yes/no question and get a mystical response.

🫂 Actions

  • /hug — Hug a friend.
  • /pat — Pat someone.
  • /slap — Slap someone.
  • /kiss — Kiss the homie.
  • /poke — Poke poke... Hey you've poked someone.
  • /bonk — BONK!
  • /cuddle — Cuddle buddies!

  • /boop — :O you booped me!
