📖 Help & Commands

⚙️ Admin & Moderation

  • sync — Sync slash commands.
  • warn <user> [reason] — Warn a user.
  • kick <user> [reason] — Kick a member.
  • ban <user> [reason] — Ban a user.
  • unban <user_id> — Unban someone by ID.
  • unwarn <user> — Remove the latest warning.
  • warnings <user> — Show all warnings for a user.
  • silence <user> <time> — Timeout a member (e.g., 10m, 1h).
  • unsilence <user> — Remove timeout from a member.
  • setwelcome <#channel> — Set the welcome message channel.
  • setgoodbye <#channel> — Set the goodbye message channel.
  • setautorole <role> — Set autorole for new members.
  • autorole — View the current autorole.
  • testwelcome — Simulate the welcome message.
  • testgoodbye — Simulate the goodbye message.
  • addreactionrole <message_id> <emoji> <role> — Add a reaction role.
  • removereactionrole <message_id> <emoji> — Remove a reaction role.
  • setlevelrole <level> <role> — Assign a role when users level up.
  • removelevelrole <level> — Remove a level-up role.
  • listlevelroles — Show all level role rewards.
